## 🇬🇧 English

# Quack & Click

**Quack & Click** is a browser-based incremental clicker game where every click brings you closer to building the ultimate duck empire.

Start with a single duck.
Click.
Earn ducks.
Buy upgrades.
Unlock buffs.
Increase your multipliers.
And watch the numbers become completely absurd.

Inspired by classic incremental games like Cookie Clicker, Quack & Click focuses on simple but addictive gameplay, mixed with a fun and slightly chaotic duck-themed atmosphere.

---

## 🎮 Features

* 🦆 Incremental clicker gameplay
* ⚡ Multipliers and progression system
* 🍞 Food-based upgrades and buffs
* 📈 Endless scaling numbers
* 🌐 Fully browser-based
* 🎨 Simple and expandable architecture
* 👥 Developed collaboratively using GitHub workflow

---

## 🛠 Technologies

* HTML5
* CSS3
* JavaScript
* JSON

## 🚧 Current Status

The game is currently in early development.

Core systems and base architecture are being built before implementing advanced gameplay mechanics, visual polish, progression systems, and additional content.

**Quack & Click** es un juego incremental de navegador donde cada click te acerca un poco más a construir el imperio definitivo de patos.
